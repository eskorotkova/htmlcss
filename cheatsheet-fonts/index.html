Import Google Fonts

It is possible to import Google Fonts directly from a Web by using @import instead of @font-face as well as by using tag <link href="">

To import one font

@import code tag <link> code
@import url('https://fonts.googleapis.com/css2?family=Raleway&display=swap'); <link href="https://fonts.googleapis.com/css2?family=Raleway&display=swap" rel="stylesheet">

To import two and more fonts

@import code tag <link> code
@import url('https://fonts.googleapis.com/css2?family=Nunito+Sans:ital,opsz,wght@0,6..12,400;1,6..12,700&family=Raleway&display=swap'); <link href="https://fonts.googleapis.com/css2?family=Nunito+Sans:ital,opsz,wght@0,6..12,400;1,6..12,700&family=Raleway&display=swap" rel="stylesheet">
